# Main Agent 电子宠物使用手册

## 一、它是什么？

你的"电子宠物"是一个 **AI 投研代理**，它会帮你做股票分析、生成交易计划、复盘决策。像素小人的动画和心情会实时反映它正在做什么。

---

## 二、快速上手

| 步骤 | 操作 | 宠物状态 |
|------|------|----------|
| **1. 创建组合** | 页面左上角"创建组合"按钮，取个名字，选模式（推荐 Paper 虚拟盘） | `待命中 (idle)` |
| **2. 添加关注** | 在右侧"自选股"区域搜索并添加你想关注的股票 | `待命中` |
| **3. 手动决策** | 点击 **"手动 Run"** 按钮，触发一次完整的决策周期 | `思考中 (thinking)` |
| **4. 查看结果** | 决策完成后，查看"交易计划"和"最近动作"面板 | `备战中 (battle)` |
| **5. 训练验证** | 点击 **"Quick Smoke"** 按钮，验证决策质量 | `训练中 (training)` |

> **如果一直显示"待命中"，说明还没有点过"手动 Run"。** 宠物需要你下达第一次指令才会开始工作。

## 三、宠物的 5 种心情

| 心情 | 像素颜色 | 动画速度 | 触发条件 |
|------|---------|---------|---------|
| **idle（待命）** | 正常 | 慢（1.2s） | 没有任何任务在执行 |
| **thinking（思考）** | 蓝色眼睛 | 中（0.6s） | Brain Run 正在执行中 |
| **training（训练）** | 绿色眼睛 | 快（0.4s） | 验证套件正在运行 |
| **battle（备战）** | 红色眼睛 | 很快（0.3s） | 有持仓 或 有待执行的交易计划 |
| **drawdown（回撤）** | 灰蓝眼睛 | 很慢（1.6s） | 组合收益为负 |

心情优先级（从高到低）：`thinking > training > drawdown > battle > idle`

## 四、核心功能

### 4.1 手动 Run（决策周期）

点击后宠物进入 thinking 状态，自动完成 6 个步骤：

1. **筛选候选股** — 从你的自选股 + 量化筛选中挑选标的
2. **扫描信号** — 检查关注的关键词/事件是否触发
3. **逐一分析** — 对每只候选股进行技术面、基本面、资讯面分析
4. **生成摘要** — 汇总分析结果
5. **LLM 决策** — AI 综合判断，生成买/卖/持有建议
6. **输出计划** — 生成交易计划和操作建议

执行期间页面每 3 秒轮询一次进度，可以看到实时状态。通常耗时 30 秒 ~ 2 分钟。

### 4.2 训练验证

- **Quick Smoke**：30-60 秒快速验证，检查决策流程是否跑通
- **完整训练**：更全面的验证 + 短窗口回测，生成 pass/warn/fail 评估

训练结果在"训练"Tab 中展示。

### 4.3 战斗（Battle Tab）

当宠物有持仓或待执行计划时进入 battle 状态。Battle Tab 显示：

- 当前持仓情况
- 待执行的交易计划
- 出战准备状态

## 五、页面布局

### 顶部

- 组合切换下拉
- "创建组合"按钮
- "手动 Run" / "Quick Smoke" 按钮

### 左侧 — 宠物 + 策略大脑

| 面板 | 内容 |
|------|------|
| **宠物主舞台** | 像素动画 + 状态描述 + 训练快照 + 出战状态 |
| **记忆规则** | 宠物从历次决策中学到的交易规则（附置信度） |
| **反思记录** | 每日/每周复盘总结，包含胜率、盈亏统计 |
| **策略演进** | 每次决策后的策略变化时间线（仓位、行业偏好、风险提醒） |
| **自选股** | 手动添加的关注标的，Brain Run 时会优先分析 |

### 右侧 — Tab 面板

| Tab | 内容 |
|-----|------|
| **宠物** | 代理状态 + 决策运行记录 |
| **训练** | 验证套件结果 |
| **战斗** | 持仓 + 交易计划 |
| **回测** | 历史回测报告 |

## 六、组合管理

- **Paper 模式（虚拟盘）**：推荐新手使用，所有交易都是模拟的
- **Live 模式（实盘映射）**：映射真实账户数据
- 可以创建多个组合，每个组合有独立的持仓、记忆和决策历史
- 切换组合：页面顶部下拉选择

---

## 七、典型使用流程

```
创建组合 (Paper)
    ↓
添加自选股 (2-3 只)
    ↓
点击 "手动 Run"  →  宠物 thinking 💭
    ↓
等待决策完成 (30s-2min)  →  宠物 battle ⚔️
    ↓
查看交易计划和分析
    ↓
点击 "Quick Smoke"  →  宠物 training 🏋️
    ↓
查看训练评估 (pass/warn/fail)
    ↓
重复：下次有新想法时再次 Run
```

宠物从第一次 Run 开始就会积累记忆，后续决策会越来越"聪明"。

## 八、常见问题

**Q: 宠物一直"待命中"怎么办？**
A: 需要点一次"手动 Run"，它才会开始工作。

**Q: 自选股添加了但 Run 之后没分析？**
A: 确认自选股是在当前组合下添加的，不同组合的自选股是独立的。

**Q: 反思记录 / 策略演进为空？**
A: 这些需要经过多次 Run + 复盘后才会逐渐积累。第一次 Run 后会开始生成。

**Q: Quick Smoke 和完整训练有什么区别？**
A: Smoke 只做流程验证（30-60s），完整训练还会做回测分析，耗时更长但结果更详细。

**Q: 可以同时管理多个组合吗？**
A: 可以。每个组合完全独立，通过顶部下拉切换。
